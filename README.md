# 📱 SocialMediaApp 

---

## 🚀 Features 

*   **🔒 Secure Authentication:** User signup, login, aur JWT  ke sath secure authentication.
*   **📝 Post Creation:** Users text, images, aur videos share kar sakte hain.
*   **💬 Real-time Chat:** Dosto ke sath instant messaging aur group chats.
*   **👥 User Profiles:** Custom bio, profile picture, aur follower/following ka system.
*   **🔔 Notifications:** Likes, comments, aur new followers ke liye instant alerts.

---

## 🛠️ Tech Stack 

*   **Frontend:** React.js 
*   **Backend:** Node.js / Express.js 
*   **Database:** MongoDB 
*   **Real-time:** Socket.io / WebSockets
